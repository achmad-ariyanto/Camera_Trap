FTPS - Securing FTP with TLS

FTPS is a secure implementation of FTP, whereby the FTP commands and the data are transmitted over secure channels via SSL/TLS.

FTPS is specified by RFC4217, "Securing FTP with TLS" by Ford-Hutchinson, et al. RFC4217 is an implementation of RFC2228, "FTP Security Extension" using TLS. RFC2228 is a generic specification which is designed to be applicable to a wide range of security protocols. Ford-Hutchinson uses appropriate elements of the RFC2228 to secure FTP via TLS.
