Feature List

  • Secure FTP file transfers using SSL/TLS. SSL 3.0, and TLS 1.0, 1.1 and 1.2 are supported. The library supports FTPS (implicit and explicit mode) and is compatible with most secure FTP servers.
  • Secure FTP file transfers using SSH. The library supports SFTP and is compatible with most SFTP servers.
  • Secure FTP file transfers using SCP.
  • Multiple File Transfers (including Directory Transfers) supported.
  • Multi-Protocol Client that permits switching protocols with a single method call.
  • Multiple, simultaneous transfers by use of FTP connection pools.
  • Background transfers via a comprehensive asynchronous API.
  • Multi-threaded performance without multi-threaded programming.
  • FXP server-to-server transfers supported for FTP.
  • No runtime licensing or royalties.
  • All major FTP operations are supported.
  • FTP/FTPS through IPv6.
  • Connects to FTP servers through Proxies (all versions) and HTTP Proxies.
  • Secure data and control channels.
  • Supports server validation.
  • Supports standard OpenSSL certificates (including certificates issued by CAs).
  • Supports DSA and RSA keys for SFTP, and supports all common public and private key formats.
  • Supports zlib compression for SFTP file transfers.
  • The same simple, intuitive API as the freely available edtFTPj, meaning upgrades to secure transfers via the SSL version are very easy.
  • Progress events, permitting notification of the progress of file transfers.
  • Transfers file in ASCII and binary modes.
  • Scripting engine for running FTP scripts.
  • FTPInputStream and FTPOutputStream.
  • SSHFTPInputStream and SSHFTPOutputStream.
  • Supports transfer interruption and resumption, including resumption at a user-specified offset for downloads.
  • Transfers data directly to and from memory.
  • Can execute remote SITE commands.
  • Appends data to existing server files.
  • Advanced logging capabilities, including optional integration with log4j.
  • Source code is optionally available for purchase to protect your investment.
